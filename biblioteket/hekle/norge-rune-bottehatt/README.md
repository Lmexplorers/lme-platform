# NORGE/NORWAY runehatt (heklet)

Rød heklet bøttehatt i staver, med stripet brem i rødt, hvitt og blått og
"NORGE"/"NORWAY" heklet på i grov, håndhugget runestil på én linje. Lite norsk
flagg på toppen. Størrelser fra barn til herre.

Bygg: `python3 build_rune.py no` og `python3 build_rune.py en` gir
`hekle_rune.html` / `hekle_rune_en.html`. Rendres til PDF med Chromium
(print-to-pdf). Fontene ligger i `fonts/`.
