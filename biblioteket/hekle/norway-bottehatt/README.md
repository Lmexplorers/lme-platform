# NORWAY-bøttehatt, HEKLEoppskrift (LME-stil)

`NORWAY-hekleoppskrift-LME.pdf` (9 sider). Renates hekleoppskrift for NORWAY
Bucket Hat, skrevet inn i LME-malen: topp (magisk ring), sider med NORWAY heklet
inn (tapestry), brem, flaggstriper og bølget kant. Drops Paris, heklenål 5 mm.
Bygg på nytt: `python3 build_hekle.py` så headless chromium til PDF.
