Alt redigeres ett sted

Du trenger ikke åpne HTML-en. All tekst, pris, nedtelling og lenker styres fra én enkelt fil — bare endre verdiene og lagre.

  • ✓Overskrifter, brødtekst og knappetekster på alle fire sider
  • ✓Pris, «før»-pris og antall timer på nedtellingen
  • ✓Lenker til e-postliste, betaling og videresending
// funnel-config.js
tilbudPris: {
  belop: 99,
  valuta: "kr",
  visningFor: "299 kr",
  timerNedtelling: 72
},
brand: {
  checkoutUrl: "", // din betalingslenke
  etterKjop: "takk-kjoper.html"
}
