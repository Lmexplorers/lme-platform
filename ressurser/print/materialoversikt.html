Materielloversikt

Hva som finnes i hvert område, kort fortalt. Materiellet går fra konkret til abstrakt, og fra enkelt til sammensatt.
Merk: Dette materiellet selges ikke hos LME. Oversikten er en faglig veiledning, og det fysiske Montessorimateriellet kjøper du hos egne leverandører av Montessorimateriell.

Praktisk liv

  • Helle og øse mellom kanner og skåler.
  • Kneppe og knytte med knapper, glidelåser og sløyfer.
  • Feie og tørke opp etter seg selv.
  • Dekke bord og rydde det av igjen.

Sansetrening

  • Det rosa tårnet, som lærer størrelse og balanse.
  • Den brune trappa, som lærer tykt og tynt.
  • Fargebrett for å pare og gradere farger.
  • Lydbokser for å kjenne igjen og pare lyder.
  • Geometriske former å kjenne på og benevne.
  • Sylinderblokkene som trener øye, hånd og dimensjon.
  • De geometriske solidene å kjenne igjen med hendene.
  • Binomisk og trinomisk kube som forbereder algebraen.

Språk

  • Sandpapirbokstaver som barnet følger med fingeren, også med æ, ø og å.
  • Sandpapirbokstaver for dobbeltlyder og lydkombinasjoner.
  • Det bevegelige alfabetet i treboks for å bygge ord.
  • Lydleker som trener å høre lyder i ord.
  • Lesekort som knytter ord til bilder og ting.
  • Metallinnsatsene som forbereder skrivehånden.
  • Klassifiseringskort som bygger ordforrådet.
  • Ordklassesymbolene, også i tre med eget brett, som gjør grammatikken konkret.
  • Grammatikkbokser for å utforske ordklassene.
  • Setningsanalyse med sirkler og piler.

Matematikk

  • Tellestaver som viser mengde og lengde.
  • Sandpapirtall som barnet følger med fingeren.
  • Spolekasser som knytter tall til mengde.
  • Tall og brikker for partall og oddetall.
  • Perlemateriell for å telle og regne konkret.
  • Det gylne perlemateriellet for titallssystemet.
  • Seguin-brettene for tallene 11 til 99.
  • Hundrebrettet og perlekjedene for telling og hopptelling.
  • Store tallkort fra 1 til 9000 for å bygge store tall.
  • Kulerammene, liten og stor, på veien mot abstrakt regning.
  • Frimerkespillet for de fire regneartene.
  • Multiplikasjonsbrettet og divisjonsbrettet med perler.
  • Brøksirklene som gjør brøk konkret.

Kultur

  • Globus og kontinentkart for geografi.
  • Botanikk med blader, planter og puslespill.
  • Zoologi med dyr og deres deler.
  • Enkle tidslinjer som viser før og nå.
Start konkret. Abstraksjonen kommer når barnet er klart.
